The algorithm name.

Returns
the algorithm name

StaticAlgorithmName returns the algorithm's name as a static member function.

This is the IETF's variant of Bernstein's ChaCha from RFC 8439. IETF ChaCha is called ChaChaTLS in the Crypto++ library. It is _slightly_ different from Bernstein's implementation.
