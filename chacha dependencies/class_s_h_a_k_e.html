Detailed Description

SHAKE message digest base class.

SHAKE is the base class for SHAKE128 and SHAKE258. Library users should instantiate a derived class, and only use SHAKE as a base class reference or pointer.

See also
Keccak, SHA3, SHAKE128, SHAKE256, FIPS 202, SHA-3 Standard: Permutation-Based Hash and Extendable-Output Functions
Since
Crypto++ 8.1
