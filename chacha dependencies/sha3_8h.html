Detailed Description

Classes for SHA3 message digests.

The Crypto++ implementation conforms to the FIPS 202 version of SHA3 using F1600 with XOF d=0x06. Previous behavior (XOF d=0x01) is available in Keccak classes.

See also
SHA-3, SHA-3 STANDARD (FIPS 202).
